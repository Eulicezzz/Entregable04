Proyecto: Sistema de Gestión Municipal (Yau)
Descripción: Automatización de gestión documental y priorización de trámites.

Tecnologías: Backend (Python/ML), Frontend (React/JS).

Cómo empezar:

Clonar el repositorio.

Instalar dependencias backend: cd backend && pip install -r requirements.txt.

Instalar dependencias frontend: cd frontend && npm install.
